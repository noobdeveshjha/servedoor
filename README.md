# Servedoor - Swiggy Clone

A full-stack food delivery app like Swiggy built with React Native + Expo.

## Features
- Restaurant browsing
- Cart system
- Order placement
- Real-time tracking (coming soon)

## Tech Stack
- React Native
- Expo
- Zustand
- Tailwind

## Setup
```bash
git clone https://github.com/noobdeveshjha/servedoor.git
cd servedoor
npm install
npx expo start
```
